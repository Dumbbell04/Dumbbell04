Project 5: 가상 회사 데이터 대시보드 제작 프로젝트

구상 이유

가상의 회사 데이터를 기반으로 매출, 비용, 고객 데이터를 시각화하고 인사이트를 도출해보는 실습을 통해 데이터 대시보드 제작 경험을 쌓기 위해 기획했습니다.

목표

매출, 고객 수, 비용 등의 데이터를 기반으로 Power BI 또는 Tableau를 활용해 실시간 데이터 대시보드 제작 및 분석 리포트 작성

진로에 도움이 되는 점

컨설팅 업무에서 자주 사용하는 데이터 시각화 도구 실습과 비즈니스 인사이트 도출 경험을 통해 실무 감각을 키울 수 있을 것입니다.
